# Bloom — Multi-Page Business Website Template

A fully responsive, multi-page business website template built with pure HTML and CSS. Designed with a dark luxury aesthetic — no frameworks, no dependencies, no backend required.

> **This is a sample/demo project.** All content (business name, menu items, prices, testimonials) is placeholder data. The template can be fully customised to suit any business — café, salon, restaurant, clinic, or any local service provider.

## Pages
- **Home** — Hero, quick navigation cards, about section, services, testimonials, CTA
- **Menu** — Category-wise item listing with prices and tags
- **Gallery** — Masonry grid layout with filter buttons
- **Order** — WhatsApp-integrated order form (no backend needed)
- **Contact** — Enquiry form, business hours, location, and Google Maps placeholder

## Features
- Dark gold luxury theme (`#c9a96e` accent)
- Playfair Display + Inter typography
- Glassmorphism fixed navbar with blur
- Fully mobile responsive
- Zero external dependencies (Google Fonts only)
- WhatsApp order button with pre-filled message

## Customisation
Replace the following to adapt for any client:
- Business name, logo, and tagline
- Menu items and prices
- Phone number in `order.html` (`wa.me/91XXXXXXXXXX`)
- Google Maps embed in `contact.html`
- Placeholder emojis with real images
